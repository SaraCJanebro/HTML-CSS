Itálico,para dar ênfase aos textos

para dar ênfase usar a tag em

Para texto marcado

Usar a tag mark. Se eu quiser mudar a cor do marcador tem que usar parâmetro da CSS posso fazer somente na tag

Ou posso definir a cor para todas as marcações na head do código

Texto grande e texto pequeno

Usar a tag big para texto maior que o padrão

Usar a tag small para texto menor que o padrão

Textos deletados e textos inseridos

Usar a tag del para textos que não devem ser considerados, mas sim lidos

Usar a tag ins para textos que foram colocados depois
